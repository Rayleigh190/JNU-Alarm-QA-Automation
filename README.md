## 📖 Description
전대알림 앱에는 140개 이상의 알림 설정이 있습니다. 주기적으로 알림 설정을 테스트해야 하지만 단순하고 반복적인 테스트 케이스가 많아 테스트 자동화를 개발했습니다.

## 🕹 Function
- 학사/장학 알림 설정 테스트
- 단과대 알림 설정 테스트
- 학과 알림 설정 테스트
- 전문대학원 알림 설정 테스트
- 특수대학원 알림 설정 테스트
- 사업단 알림 설정 테스트
